Define what gets displayed in the getDom() function.
May use variables stored in the module object.
https://docs.magicmirror.builders/development/core-module-file.html#getdom

To update the display, call the this.updateDom() function.

General code structure:

Config variables
api key
API update frequency - default 30 minutes
active hours start
active hours end
Active days



getDom()
    If during active hours
        return DOM object that shows bus times
    else
        Return DOM object that shows module inactive

function active_hours()
    return true if we are in active hours

function next_bus_time (Transit API result, current time)
    return [Time of next bus, Time to catch bus, Time of bus after the next, Time to catch bus]


function fetch_bus_times()
    return the next 10 bus times from transit

bus_times = fetch_bus_times()
store current time as transit_update_time

while (true)
    if active_hours
    Update the 4 variables to display
    e.g., next_bustime = next_bus_time()[0]

    update display via this.updateDom()

    if active_hours
        if current_time - transit_update_time > configured interval
            bus_times = fetch_bus_times()
            store current time as transit_update_time

    sleep 10 seconds
